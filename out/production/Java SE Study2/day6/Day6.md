## 数组
- 相同类型数据的集合叫做数组
- 如何定义数组
  - type [] 变量名=new type[长度]；
  - type 变量名[]=new type[长度]； 
  - tye []变量名={a,b,c,}
  
- 名为length的属性, public final属性
- int 数组值默认为0
- bool 为false
- equals 是判断引用，而非里面的值
- 引用数据类型数组存放的是对象的引用，而非对象


## 二维数组
二维数组是一种平面的二维结构，本质上一种
- 定义方式 type [][] 变量名=new type[m][n]
- Java支持不规则的二维数组，
- int [][]s=new int [][]{{1,2,3},{4,5,6}};

### 三维数组
- 定义方式 type [][][] 变量名=new type[m][n][p]
